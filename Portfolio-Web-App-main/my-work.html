Victorious Volleys

The culminating project I am working is a game called Victorious Volleys, which game like Pong but with more features. The game has two players, and the goal are for the players to hit the ball on to the otherside. It has dash line in the middle to ensure the players don't go to the otherside. The game is up to 10 and it shows the win streak of the players. It also has power up feature that makes the other player freeze until a goal is scored.

The game was created using the JavaFX library and the Scene Builder editor. The Scene Builder editor allows for easy drag-and-drop objects and edits to the object without needing to code the colour in Java. The objects that are dragged and dropped into the Anchor Pane can be assigned IDs, which allow the person to change and make edits to the object in the Java class files. Simply type @FXML private "the object" and "name you want to call the object." How the rectangle moves is by creating boolean values for the key presses, which will turn true when they are pressed. This is used to meet the conditional if statement that makes the player actually move. In the if statements, it gets the layout of the rectangle and adds or subtracts the X or Y value to make it either right, left, up, or down. The game ball was made by getting the layout of the ball and adding an integer value to the layout so that it moved. All of this is in an animation loop. If the ball has top and bottom borders, it bounces by multiplying the y movement of the ball by -1. If the ball hits the player, it multiplies by -1.5 for the circle's x movement to bounce off the player and increase the speed. The game gives the players points by having the ball pass a certain X value, which is basically when the ball passes the borders of the left or right sides. If the player reaches 10 points, the game ends by stopping the ball, making the win streak labels, and clicking the play again button to show the object as visible using setVisible(true). Additionally, a power feature was added, so when the first player reaches 5 points, the other player has a power up on their side, which freezes the other player and only unfreezes when the losing player scores a point. The win streak feature was created by creating functions. One function writes the number of wins a player has individually in a txt file, and the other function reads that file, gets the number, and returns it so that the win streaks are saved even after closing the game. The buttons were made by using an action event, so when the person clicks the button, it does whatever is in the function, the play again resets the scores, and the timeline starts again to make the ball again. The back button just controls the scene by, like, targeting the FXML that you want to change to.
