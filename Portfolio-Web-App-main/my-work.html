Victorious Volleys

The Culminating Project I am working on is Pong like game Victorious Volleys, but with more features. The game has two rectangles you can move using the keyboard one rectangle controls being WASD and the other being TFGH. They can move in both the x and y direction, but cannot go out of the screen. There is a center dash line that prevents the rectangles from going to each other side. There is ball that the rectangles to bounce of them to the other player which increase the speed of the ball. The ball does bounce on the top and bottom screen borders, but it doesn't increase the speed. If the ball goes out of the screen on the right or left it is a point for one of the players. Additonally there is power up that comes into the game which will slow the other player down. The game is to 10, and shows who wins, what there win streak is, and shows the play again button.

I created the game through Java using JavaFX library to make it easier to build the game. I made IDs with JavaFX, so I can link lines of code to the objects in my SceneBuilder which is an easy way to make scenes and objects. The AnchorPane in SceneBuilder needs to be linked to a class file to get the IDs made and the code of the class. I made boolean variables for keys, so that when a key was pressed it became true making it meet the conditions for moving the players. The movement of the players was made by getting where they were on the screen using getLayout function and then add or subtract depending which way I wanted to move the player. For example, to move the player down I would add to the x value of getLayoutX of the players.
